Convert a map to vector of key-value pairs in C++

In this post, we will discuss how to convert a map to vector of key-value pairs in C++. In other words, construct a std::vector<std::pair<K,V>> from std::unordered_map<K,V> or std::map<K,V>

 

1. Vector Constructor

The simplest and the most efficient solution would be to get the iterators to the beginning and ending of the given map and pass them to the range constructor of vector class. This will construct a vector of key-value pairs in the same order as present in the map.

template<typename K, typename V>
std::vector<std::pair<K,V>> mapToVector(const std::unordered_map<K,V> &map)
{
    return std::vector<std::pair<K,V>>(map.begin(), map.end());
}

2. Using std::copy

If we already have a vector with enough space to hold all entries of the map, the recommended solution is to use the standard algorithm std::copy from <algorithm> header. This requires an input range and the output iterator to the destination vector.

template<typename K, typename V>
std::vector<std::pair<K,V>> mapToVector(const std::unordered_map<K,V> &map)
{
    std::vector<std::pair<K,V>> v;
    v.reserve(map.size());
 
    std::copy(map.begin(), map.end(), v.begin());
 
    return v;
}

{
    std::vector<std::pair<K,V>> v;
    std::copy(map.begin(), map.end(), std::back_inserter(v));
 
    return v;
}

3. Using std::transform

Another alternative is to use std::transform algorithm defined in <algorithm> header. It applies a given operation to each of the elements in the given range and stores the result in another range.

template<typename K, typename V>
std::vector<std::pair<K,V>> mapToVector(const std::unordered_map<K,V> &map)
{
    std::vector<std::pair<K,V>> v;
 
    std::transform(map.begin(), map.end(),
                std::back_inserter(v),
                [](const std::pair<K,V> &p) {
                    return p;
                });
 
    return v;
}

4. std::for_each

We can also use std::for_each algorithm that applies the given operation to each of the elements in the given range. To convert a map to vector of key-value pairs, the range should be iterators to the beginning and ending of the given map and operation should be push_back() to insert each entry into the vector.

template<typename K, typename V>
std::vector<std::pair<K,V>> mapToVector(const std::unordered_map<K,V> &map)
{
    std::vector<std::pair<K,V>> v;
 
    std::for_each(map.begin(), map.end(),
                [&v] (const std::pair<K,V> &entry)
                {
                    v.push_back(entry);
                });
 
    return v;
}

5. Using Loop

Finally, we can also iterate the map using a simple for-loop or range-based for loop (C++11) and insert each entry to the vector as shown below:

template<typename K, typename V>
std::vector<std::pair<K,V>> mapToVector(const std::unordered_map<K,V> &map)
{
    std::vector<std::pair<K,V>> v;
 
    for (auto it = map.cbegin(); it != map.cend(); ++it) {
        v.push_back(*it);
    }
 
    return v;
}

template<typename K, typename V>
std::vector<std::pair<K,V>> mapToVector(const std::unordered_map<K,V> &map)
{
    std::vector<std::pair<K,V>> v;
 
    for (const std::pair<K,V> &i: map) {
        v.push_back(i);
    }
 
    return v;
